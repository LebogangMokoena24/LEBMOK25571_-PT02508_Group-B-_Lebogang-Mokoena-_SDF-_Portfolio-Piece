# LEBMOK25571_-PT02508_Group-B-_Lebogang-Mokoena-_SDF-_Portfolio-Piece

**Overview:** 

This project is a personal portfolio website designed to exactly duplicate a Figma template.  The website is clean and responsive, highlighting my skills, projects, education, tools, and job experience.

**Presentation links:** 

1. https://drive.google.com/file/d/1R0s48XmGwf7kQE2KAtdQ9_UvQBSFBQoh/view?usp=drive_link

2. https://youtu.be/W0ZMAv7nKQQ?si=1sn_Fm3pjLaVyODw 

**The primary goals of this project were:**

-Practice turning a design into functioning HTML and CSS.

-To create a fully responsive layout that works on both desktop and mobile devices.

-Content should be organised using semantic HTML and stylistic components to ensure maintainability.

**Features**

 -Flexbox and media queries are used in responsive design to ensure that desktop and mobile devices adapt seamlessly.

 -Sections, articles, headers, and other semantic tags make HTML more readable and accessible.

 -Project showcases: Each project includes a title, brief, technology used, and responsibilities.

 -Tools section: Displays tools and technologies as icons, organised vertically.

 -Education and experience: Clearly arranged with cards for readability.

 -Contact information is in the footer, including an email address, phone number, and a link to LinkedIn.

 **Technologies Used**

-**HTML5** – For semantic and structured content.

-**CSS3** – For styling, layout, and responsive design.

-**Flexbox & Grid** – For responsive, flexible layouts.

-**Google Fonts** (Poppins) – For clean typography.

-**Emoji icons** – Used in tools section and project hearts to simplify icon use without external kits

**Key Lessons from the project:**

 -Converting Figma designs into useful, responsive code.

 -Using semantic HTML tags (section, article, and aside) to improve structure.

 -Implementing responsive layouts with flexbox and media queries.

 -CSS classes are used to style and organise content in a way that is manageable.

 -Understanding the interaction of design, layout, and code functionality.
